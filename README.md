# Bytes.co Alpine ncdu

A no-frills Alpine ncdu container. Currently based on Alpine 3.14. Small and secure. Helpful when you run out of disk space.

## Example Usage
```bash
docker run --rm -it -v /tmp/zip:/data bytesco/pigz my-big-file
```
